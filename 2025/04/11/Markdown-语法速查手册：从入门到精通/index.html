Markdown语法详解

1、标题

使用 # 符号表示标题，# 的数量表示标题的层级（1 到 6 级）。

渲染后：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2、段落与换行

段落：段落之间需要空一行。
换行：在行尾添加两个空格。

渲染后：
这是第一段。

这是第二段。
这是同一段的换行。

3、强调

加粗：用 ** 或 __ 包裹文本。
斜体：用 * 或 _ 包裹文本。
斜体+加粗：用 * 或 _ 包裹文本。
删除线：用 ~~ 包裹文本。

渲染后：
加粗文本
斜体文本
斜体+加粗
删除线文本

4、列表

Markdown中有无序列表和有序列表两种列表

无序列表

使用 -、* 或 + 表示。

渲染后：

  • 项目1
  • 项目2
    • 子项目

有序列表

使用数字加点表示。

渲染后：

  1. 第一项
  2. 第二项
    1. 子项

5、链接与图片

链接：[显示文本](URL)。
图片：![替代文本](图片URL)。

渲染后：
百度

6、引用

使用 > 表示引用

渲染后：
> 这是一个引用
>> 这是嵌套引用

7、代码

行内代码：用 (``) 包裹。

示例：

渲染后：
这是 行内代码

代码块：用三个反引号 (```) 包裹，可指定语言。

示例：

渲染后：
def hello():
print(“hello Markdown”)

8、表格

使用 | 和 - 创建表格。

渲染后：

列1 列2 列3
数据1 数据2 数据3

9、分割线

使用至少三个或更多-或*或_创建分割线

渲染后：

tips

1、如果分割线符号少于三个，可能被解析为标题

2、使用同一种符号（如 *** 或 —），不要混用

10、任务列表

使用 - [ ] 和 - [x] 表示任务列表。

渲染后：

  • 完成任务1
  • 完成任务2
  • 完成任务3
